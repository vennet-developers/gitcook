## Installation

To use CCLI, all you need to do is install the
`@vennet/ccli` package:

```sh
# with yarn
$ yarn add @vennet/gitool -g

# with npm
$ npm i @vennet/gitool -g

# with pnpm
$ pnpm add @vennet/gitool -g

# with bun
$ bun add @vennet/gitool -g
```
